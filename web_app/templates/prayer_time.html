Today's Prayer Time Schedule Checker

Do you want to improve your prayer habit? With just an idea that it will improve myself in the long-run, i started this project. The objective was clear, I want to be aware of today's prayer time then adjust my work schedule accordingly.

But I don't stop there. I then try to calculate much closer the current time versus the closest prayer time. It just give me the time finish my work before the break. Here is how I do it.
Note: I use muslimpro for this example and set the location to Cibinong, Indonesia. 

    # import libraries to scrape data and formatting date and time
    import requests
    from bs4 import BeautifulSoup
    from datetime import datetime

    # scrape prayer time for today from web and store it in a list.
    def get_prayer_times(url):
        # Send a GET request to the URL
        response = requests.get(url)

        # check if the request was successful
        if response.status_code == 200:
            soup = BeautifulSoup(response.content, 'html.parser')

            # find the prayer time table
            prayer_times_table = soup.find('table', class_='prayer-times')
            
            # extract the prayer times for today
            current_date = datetime.now().strftime('%a %-d %b')
            today_prayer_times = []
            rows = prayer_times_table.find_all('tr')
                for row in rows:
                if row[0] == current_date:
                    today_prayer_times.append(row)
        
        else:
            "Failed to retrieve data. Check your internet connection or URL."

        return today_prayer_times
